# WorkPiece  
Web Portofolio Projek PBL  
Politeknik Negeri Batam  

## Deskripsi Proyek  
WorkPiece merupakan aplikasi berbasis web yang dikembangkan sebagai media portofolio digital untuk mendokumentasikan dan menampilkan hasil proyek Project-Based Learning (PBL) mahasiswa. Website ini memungkinkan mahasiswa untuk mengelola proyek yang telah dikerjakan secara terstruktur, mulai dari deskripsi proyek, tampilan visual, hingga media pendukung.  

Selain sebagai sarana dokumentasi, WorkPiece juga berfungsi sebagai media bagi dosen untuk melihat, memberikan komentar, serta melakukan penilaian terhadap proyek mahasiswa secara langsung melalui sistem.

## Tujuan Pengembangan  
Website WorkPiece dikembangkan dengan tujuan:  
- Menyediakan media portofolio digital bagi mahasiswa  
- Mendukung penerapan pembelajaran berbasis proyek (Project-Based Learning)  
- Mempermudah dosen dalam memantau dan menilai hasil proyek mahasiswa  
- Menyimpan dokumentasi proyek mahasiswa secara terstruktur berbasis web  

## Teknologi yang Digunakan  
- HTML  
- CSS  
- Bootstrap  
- PHP  
- MySQL  

## Fitur Utama  

### Mahasiswa  
- Login menggunakan NIM dan password  
- Mengelola profil mahasiswa  
- Menambahkan proyek PBL  
- Mengedit dan menghapus proyek  
- Melihat dan mencari portofolio mahasiswa lain  

### Dosen  
- Login dosen  
- Melihat portofolio mahasiswa  
- Memberikan komentar proyek  
- Memberikan penilaian proyek  
- Mengelola profil dosen  

## Struktur Menu  
Menu utama pada Website WorkPiece terdiri dari:  
- Beranda  
- Tentang  
- Login  

Menu akan menyesuaikan setelah pengguna berhasil login sesuai dengan hak akses.

## Alur Penggunaan Sistem  

### 1. Landing Page  
Pengguna dapat mengakses halaman Beranda dan Tentang untuk melihat informasi umum mengenai Website WorkPiece. Dari halaman ini, pengguna dapat menuju ke halaman login.

### 2. Login  
Mahasiswa login menggunakan NIM dan password, sedangkan dosen login menggunakan username dan password. Data diverifikasi melalui database dan pengguna diarahkan ke halaman utama sesuai perannya.

### 3. Halaman Mahasiswa  
Mahasiswa dapat melihat halaman home, mengelola profil, menambahkan proyek PBL, mengedit atau menghapus proyek, serta melihat portofolio mahasiswa lain.

### 4. Halaman Dosen  
Dosen dapat melihat daftar portofolio mahasiswa, membuka detail proyek, serta memberikan komentar dan penilaian terhadap proyek PBL.

## Batasan Sistem  
- Sistem difokuskan pada pengelolaan dan penampilan portofolio proyek PBL  
- Fitur disesuaikan dengan kebutuhan dasar mahasiswa dan dosen  
- Pengembangan sistem masih dapat ditingkatkan pada fitur lanjutan  

## Penutup  
Website WorkPiece dikembangkan sebagai media pendukung pembelajaran berbasis proyek yang memfasilitasi mahasiswa dalam mendokumentasikan dan menampilkan hasil karya secara digital. Sistem ini membantu proses evaluasi proyek oleh dosen serta mendukung dokumentasi PBL yang lebih terstruktur di Politeknik Negeri Batam.
